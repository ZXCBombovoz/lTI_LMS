
# LTI 1.3

Этот проект реализует интеграцию LTI 1.3 для LMS Moodle. Он оптимизирован для развертывания и использует **React** для фронтенда и **Python** для бэкенда.

## Демо-режим

1.  Откройте развернутое приложение.
2.  Нажмите кнопку **"Запустить демо"**. 
3.  Это имитирует успешный запуск из Moodle и покажет, как приложение отображает данные студента или учителя.

## Как сгенерировать RSA ключи

Для работы LTI 1.3 вам нужна пара ключей. Вы можете сгенерировать их в терминале:

```bash
# Генерируем приватный ключ
openssl genrsa -out private.pem 2048

# Конвертируем в формат PKCS8 (нужен для библиотеки jose)
openssl pkcs8 -topk8 -inform PEM -outform PEM -nocrypt -in private.pem -out private_pkcs8.pem

# Получаем публичный ключ (если нужно)
openssl rsa -in private.pem -pubout -out public.pem
```

## Настройка в Moodle

1.  Перейдите в **Администрирование сайта** > **Плагины** > **Внешние средства** > **Управление средствами**.
2.  Нажмите **"Настроить средство вручную"**.
3.  Используйте URL-адреса, которые отображаются на главной странице вашего приложения после деплоя:
    *   **URL-адрес инициирования входа**: `.../api/init`
    *   **URI перенаправления**: `.../api/launch`
    *   **URL-адрес набора ключей (JWKS)**: `.../api/jwks`
4.  Выберите **LTI 1.3** и **Keyset URL**.
5.  Сохраните. Moodle выдаст вам Client ID.

## Структура проекта

*   `/api/init.ts`: Обработка OIDC Login Initiation.
*   `/api/launch.ts`: Обработка POST запроса от Moodle и проверка JWT.
*   `/api/jwks.ts`: Предоставление публичного ключа вашего инструмента.
*   `/src/App.tsx`: Фронтенд на React.
